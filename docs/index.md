## Project Overview

Bismuth is a local-first PKM desktop application with rich note/vault workflows, planner and knowledge tooling, and native integrations via Tauri.

## Quick Reference

- **Tech Stack:** SvelteKit 2, Svelte 5, Vite 6, TypeScript 5, Tauri 2, Rust 2021
- **Entry Point (UI):** `src/routes/(app)/+layout.svelte`
- **Entry Point (Native):** `src-tauri/src/lib.rs` / `src-tauri/src/app.rs`
- **Architecture Pattern:** Layered desktop runtime with frontend/native split

## Getting Started

### Prerequisites

- Node.js 22.13+
- pnpm 11.11.0
- Rust toolchain stable

### Setup

```bash
pnpm install
```

### Run Locally

```bash
pnpm dev
pnpm tauri:dev
```

### Run Tests

```bash
pnpm test
```
