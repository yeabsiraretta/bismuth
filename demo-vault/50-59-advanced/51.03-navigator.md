# Navigator

The Navigator is a multi-pane file browser with tag trees, property browsing, and keyboard-driven navigation.

## Enabling

Enable **Navigator** in Settings > Features. It appears as a left sidebar panel.

## Features

### Multi-Pane Layout

- **Tag Tree**: Browse all tags in a collapsible hierarchy with counts
- **Property Browser**: Explore note properties and their values
- **File List**: Filtered note list with previews and metadata
- **Shortcuts**: Pin frequently-used folders, tags, and notes for quick access

### Keyboard Navigation

- **Arrow keys**: Move through the file list
- **Page Up/Down**: Jump by page
- **Home/End**: Jump to first/last item
- **Cmd+Click**: Multi-select notes
- **Cmd+F**: Toggle inline search filter

### Back/Forward Navigation

The Navigator maintains a history stack. Use the back and forward buttons to revisit previously viewed sets of notes, just like a web browser.

### Inline Search

Press `Cmd+F` within the Navigator to filter the current file list. Press `Escape` to dismiss. The search filters in real-time as you type.

> [!tip] Pinned Shortcuts
> Pin your most-used folders, tags, and notes to the Shortcuts section for one-click access.
