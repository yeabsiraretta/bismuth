# Creative Tools

Bismuth includes creative production tools beyond traditional note-taking.

## Music Studio

Enable **Music Studio** in Settings > Features (left sidebar):

- Built-in DAW (Digital Audio Workstation)
- Piano roll editor for composing melodies
- Clip-based arrangement with transport controls
- Synthesizer integration via Tone.js
- Export compositions alongside your notes

### Commands

- `Cmd+P` > "Music" to access music commands

## Media Editor

Enable **Media Editor** in Settings > Features (left sidebar):

- Photo editing with filters, crop, and adjustments
- Video trimming and basic editing
- Media files stored alongside your notes
- Inline media previews in the editor

## Spreadsheet

Enable **Spreadsheet** in Settings > Features (left sidebar):

- CSV and JSON tabular data editor
- Formula support for calculations
- Chart generation from data
- Sort, filter, and pivot operations
- Export to CSV or embed in notes

## Arbor Editor

Enable **Arbor Editor** in Settings > Features (left sidebar):

A branching note editor for non-linear writing and exploration.

- Create a root block to start
- Branch into child blocks to explore alternatives
- Undo structural changes with `Cmd+P` > "Arbor: Undo"
- Visualize your branching tree of ideas

### Commands

- **Arbor: New Root Block** — Start a new branching document
- **Arbor: Create Child Block** — Branch from the selected block
- **Arbor: Undo** — Undo the last structural change

## Longform Writing

Enable **Longform** in Settings > Features (left sidebar):

For manuscript-length projects with multiple scenes.

- Organize writing into projects with scenes
- Track word counts and progress
- Scene status workflow (draft, revision, final)
- Draft snapshots for version comparison
