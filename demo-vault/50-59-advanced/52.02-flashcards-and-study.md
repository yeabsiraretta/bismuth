# Flashcards & Study

Bismuth can extract flashcards from your notes and sync them with Anki for spaced repetition study.

## Flashcard Syntax

### Basic Cards (double colon)

```markdown
Question :: Answer
```

### Reversed Cards (triple colon)

```markdown
Term ::: Definition (creates cards in both directions)
```

### Multi-line Cards (heading + body)

```markdown
### What is the capital of France? #card

Paris is the capital of France, located on the Seine river.
```

## Study Vault Panel

Enable **Study** in Settings > Features for vault-wide flashcard review:

- Scans all notes for flashcard syntax
- Groups cards by source note
- Tracks review progress
- Supports courses and exam tracking

## Note Flashcard Panel

Enable **Flashcards** in Settings > Features for per-note review:

- Shows flashcards detected in the active note
- Quick review mode with flip-to-reveal
- Mark cards as known/unknown

## Anki Integration

Configure in Settings > Features:

- **AnkiConnect port**: Default 8765
- **Deck prefix**: Organize synced cards under a prefix
- **Auto-scan**: Automatically detect new cards on save

Use the command `Flashcards: Sync to Anki` to push cards.

## Sample Flashcards

Bismuth :: A local-first PKM editor with flashcard support

Spaced repetition ::: A learning technique that increases intervals between reviews

### What three flashcard syntaxes does Bismuth support? #card #flashcards

Double colon (basic), triple colon (reversed), and heading + body (multi-line with #card tag).
