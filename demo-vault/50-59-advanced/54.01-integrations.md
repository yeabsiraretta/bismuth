# Integrations

Bismuth connects with external tools and services to extend your workflow.

## Git Version Control

Enable **Git** in Settings > Features (right sidebar):

- Track changes to your vault with Git
- View modified, added, and deleted files
- Commit changes with custom messages
- Auto-commit on configurable intervals
- Sync with remote repositories on startup

Configure in Settings > Vault: enable Git, set commit templates, and auto-commit preferences.

## Publishing

Enable **Publishing** in Settings > Features (left sidebar):

- Publish notes as a static website
- Toggle publish flag per-note via frontmatter (`publish: true`)
- Publication dashboard shows all publishable notes
- Use the command `Publish Site` to generate your site

## RSS Reader

Enable **RSS Reader** in Settings > Features (left sidebar):

- Subscribe to RSS and Atom feeds
- Read articles inline within Bismuth
- Save articles as notes for annotation
- Organize feeds by category

## Voice Recorder

Enable **Voice Recorder** in Settings > Features (right sidebar):

- Record voice notes directly in Bismuth
- Configurable format: WebM, OGG, or MP3
- Quality presets: low, medium, high
- Transcription support for converting speech to text

Configure in Settings > Voice: format, quality, and transcription options.

## NAS Access

Enable **NAS Access** in Settings > Features (left sidebar):

- WebDAV network storage synchronization
- Access files from your NAS directly
- Sync vault data to network drives

## OCR Import

Enable **OCR Import** in Settings > Features (left sidebar):

- Import handwritten notes via optical character recognition
- Supports PDF and image files
- Optional LLM-powered text correction
- Configurable language and model paths

Configure in Settings: OCR language, model paths, and LLM correction toggle.

## Importer

Enable **Importer** in Settings > Features (left sidebar):

- Import notes from other applications
- Supports various export formats
- Preserves metadata and structure during import

## AI Agent

Enable **AI Agent** in Settings > Features (left sidebar):

- Chat with local or cloud LLM models
- Context-aware: the agent can read your current note
- Review and apply AI-suggested changes
- Configurable history depth

Configure in Settings > AI Agent: enable/disable, note context, and max history.
