# PDF & EPUB Annotator

Annotate PDF and EPUB documents directly within Bismuth, with highlights, comments, and tags stored as markdown.

## Enabling

Enable **Annotator** in Settings > Features. It appears as a left sidebar panel.

## How It Works

Annotations are stored in markdown files with special frontmatter:

```yaml
---
annotation-target: papers/research-paper.pdf
annotation-target-type: pdf
---
```

Each annotation becomes a blockquote in the markdown file:

```markdown
> [!annotation] (Page 5)
> **PREFIX**: some context before
> **HIGHLIGHT**: ==important finding here== [yellow]
> **POSTFIX**: some context after
> **COMMENT**: This supports our hypothesis
> **TAGS**: #research, #methodology
> ^ann-001
```

## Features

### Highlight Colors

Six color presets for organizing highlights:

- **Yellow** — General highlights
- **Red** — Critical or concerning points
- **Green** — Supporting evidence
- **Blue** — Definitions and key terms
- **Purple** — Questions and uncertainties
- **Orange** — Action items and follow-ups

### PDF Viewer

- Page-by-page navigation
- Zoom controls (fit width, fit page, custom)
- Dark mode for comfortable reading
- Text selection for creating highlights

### Annotation Sidebar

- Filter annotations by color
- Edit comments and tags inline
- Delete annotations
- Jump to annotation location in document

### Obsidian Annotator Compatibility

The annotation format is compatible with the Obsidian Annotator plugin, so you can use annotation notes across both apps.

## Commands

- **Annotator: Open** — Open the annotator for the active annotation note
- **Annotator: Toggle Dark Mode** — Switch reader dark mode
- **Annotator: Toggle Sidebar** — Show/hide the annotation sidebar
- **Annotator: Close** — Close the annotator panel

## Advanced URI Integration

Open the annotator via URI:

```
bismuth://annotate?filepath=papers/research-paper.pdf
```

This creates or opens the annotation note for the given PDF.
