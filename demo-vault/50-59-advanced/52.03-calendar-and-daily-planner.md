# Calendar & Daily Planner

Bismuth integrates a calendar view with daily notes, ICS feed subscriptions, and time tracking.

## Calendar Panel

Enable **Calendar** in Settings > Features (right sidebar):

- Month/week/day views with event dots
- Click a date to open or create the daily note
- Color-coded categories for visual organization
- Navigate between months with arrow keys

## Daily Notes

Enable **Daily Note** in Settings > Features:

- Press `Cmd+Shift+D` to open today's daily note
- Automatic creation from your configured template
- Links to weekly and monthly periodic notes

## Day Planner

Write timed entries in your daily notes using this syntax:

```markdown
- [ ] 09:00 - 10:00 Team standup meeting
- [ ] 10:30 - 12:00 Deep work on project
- [x] 14:00 - 15:00 Review pull requests
- [ ] 16:00 - 17:00 Write documentation
```

The Timeline View sidebar shows these entries on a vertical timeline with a live "now" indicator.

## ICS Calendar Feeds

Subscribe to external calendars (Google, iCloud, Outlook):

- Add feed URLs in the Calendar settings panel
- Events appear alongside your daily note entries
- Auto-sync on configurable intervals
- Color-code each feed for easy identification

## Time Clock

Track time spent on tasks:

- Start/stop a clock for any task
- Active clocks display as a pulsing widget
- Clock records persist across sessions
- View time logs in the Timeline sidebar
