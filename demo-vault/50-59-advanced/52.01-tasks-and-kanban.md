# Tasks & Kanban

Bismuth extracts tasks from your notes and presents them in dedicated task management views.

## Task Panel

Enable **Tasks** in Settings > Features for a vault-wide task tracker:

- Aggregates all `- [ ]` and `- [x]` items across your vault
- Filter by status, tag, due date, or priority
- Click a task to jump to the source note
- Mark tasks complete directly from the panel

### Task Syntax

```markdown
- [ ] Incomplete task
- [x] Completed task
- [ ] Task with due date 📅 2026-07-01
- [ ] High priority task ⏫
- [ ] Task with tag #project/bismuth
```

## Kanban Board

Enable **Kanban Board** in Settings > Features for a visual board view:

- Drag and drop tasks between columns
- Columns auto-generated from task status or tags
- Group by priority, status, or custom property
- Visual progress tracking at a glance

> [!tip] Combining Views
> Use the Task Panel for quick filtering and the Kanban Board for visual planning. Both views stay synchronized.
