# Writing Lint & Speed Reader

Two features for improving your writing and consuming content faster.

## Writing Lint Panel

Enable **Writing Lint** in Settings > Features (right sidebar):

- Real-time writing style analysis
- Detects passive voice, adverb overuse, and sentence complexity
- Readability scores (Flesch-Kincaid, Coleman-Liau)
- Highlights problem areas in your text
- Suggestions for clearer, more concise prose

## Speed Reader (RSVP)

Enable **Speed Reader** in Settings > Features (right sidebar):

RSVP (Rapid Serial Visual Presentation) displays one word at a time at your chosen speed.

### How to Use

1. Open the note you want to read
2. Run the command `Speed Reader: Start` from the command palette
3. Words appear one at a time in the focus area
4. Run `Speed Reader: Close` to dismiss

### Benefits

- Read 2-3x faster than traditional reading
- Reduces subvocalization and regression
- Ideal for reviewing your own notes quickly

> [!tip] Keyboard Shortcut
> Assign a hotkey in Settings > Hotkeys for quick speed reader access.

## Versioning & Dataview

### Versioning

Enable **Versioning** in Settings > Features to track note history:

- Automatic version snapshots on save
- View version timeline with diffs
- Restore any previous version
- Configurable retention count (default: 50 versions)

### Dataview

Enable **Dataview** in Settings > Features for database-style queries:

- Query your vault using DQL (Dataview Query Language)
- Create tables, lists, and task views from note metadata
- Filter and sort by frontmatter properties
