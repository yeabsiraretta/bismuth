# Experimental Features

These features are experimental and may change significantly between releases.

## Gym Tracker

Enable **Gym Tracker** in Settings > Features (left sidebar):

- Log workouts with exercises, sets, reps, and weight
- Track nutrition and macros
- View progress over time
- Workout history stored alongside your notes

## Pokemon Tools

Enable **Pokemon Tools** in Settings > Features (left sidebar):

- Damage calculator for competitive battles
- Team builder with type coverage analysis
- Quick reference for type matchups
- Stored in your vault for easy access

> [!warning] Experimental
> These features are under active development. Their interfaces and data formats may change.

## Capture Inbox

Enable **Capture Inbox** in Settings > Features (left sidebar):

A quick capture dashboard for rapid note entry:

- Capture fleeting thoughts without switching context
- Batch-process captured notes into permanent locations
- Tag and categorize captures for later review
- Keyboard-driven for fast entry

## Changelog

Enable **Changelog** in Settings > Features (left sidebar):

Automatic vault change history:

- Tracks recently modified files
- Configurable update frequency and file limits
- Uses wikilinks for easy navigation
- Exclude folders from tracking

Configure in Settings > Changelog: path, format, max files, and excluded folders.
