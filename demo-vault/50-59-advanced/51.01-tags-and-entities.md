# Tags & Entities

Bismuth provides two complementary systems for organizing and classifying your notes beyond folder structure.

## Tags

Tags are lightweight labels you attach to notes via frontmatter or inline `#hashtags`.

### Using Tags

- **Frontmatter tags**: Add a `tags` array in your note's YAML header
- **Inline tags**: Write `#topic` anywhere in your note body
- **Nested tags**: Use `/` for hierarchy: `#project/bismuth`, `#status/active`

### Tag Browser Panel

Enable **Tags** in Settings > Features to get a dedicated sidebar panel:

- Browse all tags in a hierarchical tree
- See tag counts at a glance
- Click a tag to filter notes
- Sort alphabetically or by frequency

> [!tip] Quick Access
> Enable the Tags panel in Settings > Features, then click the tag icon in the left sidebar.

## Entities

Entities are semantic concepts extracted from your notes: people, places, organizations, tools, and ideas.

### Entity Browser

Enable **Entities** in Settings > Features for the left sidebar panel:

- Browse all detected entities grouped by type
- Click an entity to see all notes that reference it
- Filter by entity type (person, place, concept, etc.)

### Entity Detail Panel

Enable **Entity Detail** in Settings > Features for the right sidebar:

- View full entity profile when selected
- See all connections and backlinks for that entity
- Jump to any referencing note
