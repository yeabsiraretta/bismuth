# Backlinks & Connections

Bismuth tracks how your notes link to each other and can discover semantic relationships.

## Backlinks Panel

Enable **Backlinks** in Settings > Features to see which notes link to the current one.

- Every `[[wikilink]]` creates a two-way connection
- Backlinks update automatically as you edit
- Click any backlink to jump to the linking note

## Outgoing Links Panel

Enable **Outgoing Links** in Settings > Features to see what the current note links to.

- Lists all `[[wikilinks]]` in the active note
- Shows broken links (targets that don't exist yet)
- Click to navigate or create missing notes

## Semantic Connections

Enable **Connections** in Settings > Features for AI-powered similarity:

- Finds notes related by content, not just explicit links
- Uses vector embeddings to measure semantic similarity
- Discovers connections you might have missed
- Requires embeddings to be enabled in Settings > Embeddings

## Auto-Linker (Wikilink Suggestions)

Enable **Auto-Linker** in Settings > Features to find unlinked mentions:

- Scans your note for text that matches other note titles
- Suggests `[[wikilinks]]` you could add
- One-click to insert the link
- Helps build your knowledge graph organically
