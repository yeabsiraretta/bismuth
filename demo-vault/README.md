# Bismuth Demo Vault

Welcome to the Bismuth demo vault! This vault demonstrates the key features of Bismuth PKM Editor.

## What's Inside

This demo vault showcases:

- **Johnny.Decimal Organization**: Hierarchical folder structure with decimal notation
- **Zettelkasten Notes**: Atomic notes with wikilinks and backlinks
- **Markdown Features**: Formatting, lists, code blocks, and more
- **Frontmatter**: YAML metadata for notes

## Folder Structure

```
demo-vault/
├── 10-19-personal/          # Personal knowledge area
│   ├── 11-notes/            # General notes
│   └── 12-journal/          # Daily journal
│
├── 20-29-projects/          # Projects area
│   ├── 21-bismuth/          # Bismuth development
│   └── 22-learning/         # Learning projects
│
└── .bismuth/                # Bismuth configuration
    └── config.json          # Vault settings
```

## Getting Started

1. Open this vault in Bismuth
2. Explore the notes in each folder
3. Try creating new notes with `Cmd+N` (Mac) or `Ctrl+N` (Windows/Linux)
4. Link notes together using `[[wikilink]]` syntax
5. View the graph to see connections

## Key Concepts

### Johnny.Decimal System

- **Areas** (10-19, 20-29): Top-level categories
- **Categories** (11, 12, 21, 22): Subcategories within areas
- **IDs** (11.01, 11.02): Individual items

### Zettelkasten Method

- **Atomic Notes**: One idea per note
- **Wikilinks**: `[[Note Title]]` to link notes
- **Backlinks**: See which notes link to current note
- **Tags**: `#tag` for thematic organization

### Frontmatter

```yaml
---
title: Note Title
date: 2026-05-26
tags: [demo, example]
---
```
