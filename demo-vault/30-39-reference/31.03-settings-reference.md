# Settings Reference

Access settings via `Cmd+,` or the [[20-29-projects/21.02-command-palette|Command Palette]].

## Editor Settings

- **Font size** — Adjustable from 12–24px
- **Line height** — 1.2–2.0
- **Tab size** — 2 or 4 spaces
- **Spell check** — Toggle on/off
- **[[11.04-markdown-guide|Markdown]]** preview mode

## Appearance

- **Theme** — Light, Dark, or Auto (follows system)
- **Accent color** — Primary UI color
- **UI Scale** — Zoom the entire interface

## Vault Settings

- **Vault path** — Where your files are stored
- **Default note location** — Where new notes are created
- **[[30-39-reference/31.04-templates|Template]]** folder — Location of note templates
- **File extensions** — Which files to index

## Graph Settings

Configure the [[20-29-projects/21.03-graph-visualization|Graph View]]:

- Center force, repulsion, link strength
- Node size, label threshold
- Show/hide node types

## Keyboard Shortcuts

See [[11.05-keyboard-shortcuts|Keyboard Shortcuts]] for a full list. Most are customizable in settings.
