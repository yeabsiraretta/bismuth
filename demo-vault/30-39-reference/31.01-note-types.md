# Note Types

Bismuth supports several note types, each suited to different use cases.

## Markdown Notes (`.md`)

Standard text notes using [[11.04-markdown-guide|Markdown]] syntax. The backbone of your vault.

## Canvas Documents (`.canvas`)

Visual workspaces for spatial thinking. See [[40-49-canvas/41.01-self-design-workflow|Self-Design Workflow]].

## Base Notes (`.base`)

Structured data notes (tables, databases). Useful for tracking projects and collections.

## Templates

Reusable note skeletons. See [[30-39-reference/31.04-templates|Templates Reference]].

## Choosing a Note Type

| Need | Type | Example |
|------|------|---------|
| Write prose | Markdown | Journal entry, essay |
| Organize visually | Canvas | Mind map, architecture diagram |
| Track items | Base | Reading list, project tracker |
| Repeated structure | Template | Daily note, meeting notes |

## Linking Across Types

All note types support [[wikilink|wikilinks]]:
- Markdown → Markdown: `[[other-note]]`
- Canvas → Markdown: Link nodes to notes
- Base → Markdown: Cell references

### When would you choose Canvas over Markdown? #card #reference #note-types
Use Canvas for spatial/visual thinking (mind maps, workflows, design docs); use Markdown for linear text-based notes, study content, and journal entries.
