# Templates

Templates let you create notes with predefined structure and dynamic content.

## Template Types

| Type | Description | Example |
|------|-------------|---------|
| Daily | Day-specific journal | `{{date_today}}` header |
| Weekly | Week review template | Week number, goals |
| Monthly | Month overview | Calendar, objectives |
| Custom | Any structure | Meeting notes, research |

## Template Syntax

Bismuth templates use **Handlebars** with pipe syntax:

```
# {{date_today}}

## Tasks
- [ ] Morning routine
- [ ] {{title | uppercase}}

## Notes
{{content}}
```

## Built-in Variables

| Variable | Output |
|----------|--------|
| `{{date_today}}` | Today's date |
| `{{date_now}}` | Current datetime |
| `{{sys_uuid}}` | Unique identifier |
| `{{title}}` | Note title |

## Pipe Functions

Transform values inline: `{{title | uppercase}}`, `{{name | slugify}}`, `{{text | truncate:100}}`

## Creating Templates

1. Create a `.md` file in `.bismuth/templates/`
2. Add template syntax
3. Use via [[20-29-projects/21.02-command-palette|Command Palette]] → "Insert Template"

## Flashcards

What template syntax does Bismuth use? :: Handlebars with pipe functions (e.g., `{{title | uppercase}}`)

`{{date_today}}` outputs ==today's date== in a Bismuth template.

Where are Bismuth templates stored? :: In the `.bismuth/templates/` directory as `.md` files

Pipe functions ::: Inline transformations applied to template variables like `| uppercase`, `| slugify`, `| truncate:100`

### How do you create and use a template in Bismuth? #card #reference #templates
Create a `.md` file in `.bismuth/templates/` with Handlebars syntax (`{{variable}}`), then insert via Command Palette → "Insert Template" or `Cmd+T`.
