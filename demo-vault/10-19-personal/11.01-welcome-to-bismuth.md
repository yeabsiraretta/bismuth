# Welcome to Bismuth

Bismuth is a **local-first Personal Knowledge Management (PKM) editor** that combines three powerful organizational systems:

## Three Pillars

1. **[[11.02-johnny-decimal-system|Johnny.Decimal]]** - Hierarchical file organization
2. **[[11.03-zettelkasten-method|Zettelkasten]]** - Atomic note-taking with links
3. **Lightweight Ontologies** - Semantic concept relationships

## Key Features

- ⚡ **Native Performance** - Built with Tauri and Rust
- 🔒 **Local-First** - Your data stays on your computer
- 📝 **Markdown-Based** - Plain text, future-proof
- 🎨 **Obsidian-Compatible** - Use existing themes
- 📊 **Graph Visualization** - See your knowledge connections
- 🔍 **Full-Text Search** - Find anything instantly

## Getting Started

1. Explore this demo vault
2. Create your first note with `Cmd+N`
3. Link notes together using `[[wikilinks]]`
4. View the graph to see connections
5. Organize with [[11.02-johnny-decimal-system|Johnny.Decimal]]
