# Keyboard Shortcuts

Master these shortcuts to navigate Bismuth like a pro.

## Notes

| Shortcut      | Action                                                |
| ------------- | ----------------------------------------------------- |
| `Cmd+N`       | New note                                              |
| `Cmd+O`       | Open note (via [[20-29-projects/21.02-command-palette | Command Palette]]) |
| `Cmd+S`       | Save note                                             |
| `Cmd+W`       | Close note                                            |
| `Cmd+Shift+F` | Search vault                                          |

## Navigation

| Shortcut | Action                                 |
| -------- | -------------------------------------- |
| `Cmd+P`  | [[20-29-projects/21.02-command-palette | Command Palette]] |
| `Cmd+,`  | Open Settings                          |
| `Cmd+\`  | Toggle sidebar                         |
| `Cmd+G`  | Open [[11.01-welcome-to-bismuth        | Graph View]]      |

## Editor

| Shortcut      | Action              |
| ------------- | ------------------- |
| `Cmd+B`       | Bold                |
| `Cmd+I`       | Italic              |
| `Cmd+K`       | Insert link         |
| `Cmd+Shift+K` | Insert [[wikilink]] |
| `Tab`         | Indent              |
| `Shift+Tab`   | Outdent             |

## Flashcards

What shortcut opens the Command Palette? :: `Cmd+P`

`Cmd+N` creates a ==new note== in Bismuth.

What shortcut toggles the sidebar? :: `Cmd+\`

`Cmd+Shift+F` ::: Search the entire vault

How do you insert a wikilink via keyboard? :: `Cmd+Shift+K`

### What are the 3 most essential Bismuth shortcuts to memorize? #card #shortcuts #productivity

`Cmd+P` (Command Palette — universal launcher), `Cmd+N` (new note), and `Cmd+\` (toggle sidebar). These cover navigation, creation, and workspace management.
