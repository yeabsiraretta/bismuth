# Graph Visualization

The **Graph View** renders your vault as an interactive knowledge web, showing how notes connect through [[wikilink|wikilinks]].

## How It Works

1. Each **note** becomes a node
2. Each **[[wikilink]]** becomes an edge
3. A **force simulation** positions nodes organically
4. **Connected clusters** emerge naturally

## Controls

- **Pan** — Click and drag empty space
- **Zoom** — Scroll wheel (zooms toward cursor)
- **Drag Node** — Click and drag a node to reposition
- **Click Node** — Opens the note
- **Right-click** — Context menu (open, local graph, backlinks)

## Settings

Access via the gear icon in the graph toolbar:

- **Forces** — Adjust center pull, repulsion, link strength, link distance
- **Display** — Node size, label threshold, line thickness, arrows
- **Filters** — Show/hide tags, attachments, orphans

## Force Simulation

Bismuth uses a **Barnes-Hut** algorithm (O(n log n)) for repulsion, with:

- Adaptive damping for natural settling
- Jittered initialization to break symmetry
- Configurable link distance and strength

## Use Cases

- **Discover connections** you didn't know existed
- **Find orphan notes** that need linking
- **Visualize clusters** of related topics
- **Navigate** your vault spatially
