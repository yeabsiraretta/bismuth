# Command Palette

The **Command Palette** is Bismuth's universal action launcher, accessible via `Cmd+P`.

## Features

- **Quick Open** — Type a filename to open any note
- **Commands** — Run any action (create, delete, toggle, export)
- **Fuzzy Search** — Typo-tolerant matching
- **Recents** — Recently opened notes appear first

## Usage

1. Press `Cmd+P` or click the palette icon
2. Start typing a note name or command
3. Use arrow keys to navigate
4. Press `Enter` to execute

## Commands Available

- `New Note` — Creates a note (same as [[11.05-keyboard-shortcuts|Cmd+N]])
- `Open Graph` — Shows the [[20-29-projects/21.03-graph-visualization|knowledge graph]]
- `Toggle Sidebar` — Hides/shows the sidebar
- `Open Settings` — Opens [[30-39-reference/31.03-settings-reference|Settings]]
- `Insert Template` — Inserts a [[30-39-reference/31.04-templates|template]]
