# Component Generation

Invoke the `component-gen` skill to generate Bismuth UI components with UX best practices.

## Steps

1. Invoke the skill:
   - Use the `skill` tool with SkillName: `component-gen`
   - Follow all instructions returned by the skill

2. The skill will guide you to:
   - Identify component type (form, table, nav, modal, settings, dashboard)
   - Apply UX guardrails (max 7 items, min 44px touch targets, etc.)
   - Use the correct tech stack (Svelte + TypeScript + Tailwind)
   - Keep files under 300 lines (Constitution Principle VI)
   - Follow accessibility and interaction standards

## When to Use

- Creating or building a new UI component
- Planning component architecture
- Implementing features that involve UI
